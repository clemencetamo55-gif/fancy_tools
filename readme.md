
#fancy tools


## description
fancy tools est un ensemble dee scripts et d'alias Bash destine a simplifier l'utilisation de git et de l'environnement shell.
il permet d'auuutomatiser certaines taches courantes (installation, mises a jours, gestion de branches, etc..) et d'ameloirer la prodductivite en ligne de commande.

# prerequis
- OS: Linux/Unix (teste sur ubuntu et Debian)
- Git
- Bash
- nano

# installation
1- cloner le depot: bash
git clone https://github.com/utilisateur/fancy_tools.git
cd fancy_tools

2-lancer le sscript d'installation
./installl.sh
3- redemarrer le terminal pour appliquer les changements
mise  a jour des outils avec updateFancyTools
